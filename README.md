🌐 **Live site:** https://mcgee162010.github.io/McGee-Family/

## Pages

| Page | Description |
|---|---|
| `index.html` | Home — hero, baby countdown, quick links, faith statement |
| `family.html` | Meet the family — bios, wedding, pets, hobbies, faith |
| `gallery.html` | Photo gallery — family shoot, wedding, Thailand honeymoon |
| `events.html` | Baby shower RSVP — July 25, 2026 · Formspree form + .ics |

## Design

- **Theme:** Woodland Animals — sage green, warm cream, earth brown, terracotta
- **Font:** SF Pro (system font stack)
- **Style:** Apple HIG-inspired — glass cards, backdrop blur, clean typography
- **Responsive:** Mobile-first, works on all screen sizes

## RSVP

The baby shower RSVP form uses [Formspree](https://formspree.io). After submission, the address is shared privately via email/text.
